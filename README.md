# Exploratory Data Analysis (EDA) & Business Intelligence

## Project Overview
This project focuses on Exploratory Data Analysis (EDA) and Business Intelligence using a sales dataset. The analysis was performed using Python, MySQL, and Excel to uncover business insights and support data-driven decision-making.

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- MySQL
- Microsoft Excel

## Project Tasks
- Data Cleaning & Preprocessing
- Descriptive Statistics & Univariate Analysis
- SQL Business Queries
- Multivariate Analysis (Scatter Plot, Heatmap, Pair Plot)
- Static Dashboard Mock-up

## Repository Contents
- `EDA_Report.pdf`
- `SQL_Queries_With_Results.pdf`
- `Dashboard_Mockup.pdf`
- `sales_analysis.sql`
- Python source files (`.py`)
- Dataset (`Cleaned_Sales_Dataset_SQL.csv`)
- Visualization images

## Key Insights
- Identified the top revenue-generating products.
- Determined the highest-performing product category and city.
- Analyzed monthly sales trends.
- Identified top customers based on total sales.
- Compared average order value across categories.
- Analyzed sales contribution by different age groups.
